# 前端升级计划 - 用户体验优化

## 🎯 升级目标

从用户角度全面提升Web界面的易用性、功能丰富度和交互体验。

## 🎨 设计规范

### 颜色体系
```
主色调: #1890ff (蓝色 - 专业、可信)
成功色: #52c41a (绿色 - 盈利)
警告色: #faad14 (橙色 - 注意)
危险色: #ff4d4f (红色 - 亏损/警告)
辅助色: #722ed1 (紫色 - 套利)
```

### 组件规范
- 卡片圆角: 8px
- 按钮高度: 32px (中等)
- 间距: 8px, 16px, 24px (倍数关系)
- 字体大小: 12px, 14px, 16px, 18px, 24px

### 图表规范
- 统一使用 lightweight-charts (性能优异)
- 颜色与主题一致
- 支持导出图片
- 响应式大小调整

---

## 📱 响应式断点

```typescript
xs: < 576px    // 手机竖屏
sm: >= 576px   // 手机横屏
md: >= 768px   // 平板竖屏
lg: >= 992px   // 平板横屏/小屏桌面
xl: >= 1200px  // 桌面
xxl: >= 1600px // 大屏桌面
```
