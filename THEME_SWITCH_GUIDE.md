## 🔧 技术实现

### 前端实现
```typescript
// 1. 主题状态管理
const [isDarkMode, setIsDarkMode] = useState(false);

// 2. 切换主题
const toggleTheme = () => {
  if (isDarkMode) {
    document.documentElement.classList.remove("dark-theme");
    localStorage.setItem("theme", "light");
  } else {
    document.documentElement.classList.add("dark-theme");
    localStorage.setItem("theme", "dark");
  }
};

// 3. 自动加载上次的主题
useEffect(() => {
  const saved = localStorage.getItem("theme");
  if (saved === "dark") {
    setIsDarkMode(true);
    document.documentElement.classList.add("dark-theme");
  }
}, []);
```

### CSS变量
```css
/* 亮色主题（默认） */
:root {
  --bg-color: #f0f2f5;
  --text-color: #000000d9;
  --card-bg: #ffffff;
}

/* 暗色主题 */
.dark-theme {
  --bg-color: #000000;
  --text-color: #ffffffd9;
  --card-bg: #141414;
}
```

### Ant Design配置
```typescript
<ConfigProvider
  theme={{
    algorithm: isDarkMode 
      ? theme.darkAlgorithm 
      : theme.defaultAlgorithm
  }}
>
  <App />
</ConfigProvider>
```
